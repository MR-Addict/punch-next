# 值班笔记-Next.js

值班笔记 Next.js 重构

## 1. 部署

添加环境变量：

```env
MONGODB_URI="mongodb://username:password@mongodb0.example.com:27017/"
```

编译项目:

```bash
npm run build
```

启动项目:

```bash
npm start
```
